# 🔥 DDOS-MASTER-TERMUX - El DDoS más potente para Android

## ⚡ Características
- ✅ **Modo ROOT**: 15 ataques simultáneos (DHCP, ARP, Deauth, SYN, ICMP, UDP, etc.)
- ✅ **Modo NO-ROOT**: 8 ataques básicos pero efectivos
- ✅ **Detección automática** de permisos root
- ✅ **Monitoreo en tiempo real**
- ✅ **Sin dependencias externas** (instalación automática)
- ✅ **Compatible con Termux**
